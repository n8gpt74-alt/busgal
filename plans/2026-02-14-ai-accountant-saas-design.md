# AI-Бухгалтер для малого бизнеса — Дизайн документа

## 1. Обзор проекта

### Название
**SmartAccount** (рабочее название: "БухGPT")

### Идея
AI-ассистент для бухгалтерии малого бизнеса (ИП, ООО, самозанятые), который отвечает на вопросы по налогам, помогает вести учёт и подсказывает оптимальные решения.

### Проблема
- Малый бизнес часто не может позволить себе штатного бухгалтера
- Обычные бухгалтерские программы сложны и требуют обучения
- Предприниматели тратят много времени на рутину и боятся ошибок

### Ключевое предложение
Вместо дорогого бухгалтера или сложной программы — простой чат с AI, который знает все налоговые законы и всегда подскажет, что делать.

## 3. Ключевые функции

### 3.1 AI-бухгалтер в чате
- Спроси: "какой налог платить за этот квартал?" — получишь ответ с ссылками на статьи НК РФ
- Спроси: "можно ли списать этот расход?" — получишь анализ с рисками
- Спроси: "как оформить сотрудника?" — пошаговая инструкция

### 3.2 Налоговый календарь
- Напоминания о сроках сдачи отчётов и уплаты налогов
- Push-уведомления за неделю и за день до срока

### 3.3 Классификатор операций
- Загрузи чек → AI определит статью расхода
- Подскажет, можно ли принять к вычету

### 3.4 Шаблоны документов
- Договоры, акты, счета-фактуры
- Автозаполнение на основе контекста

### 3.5 Функции для самозанятых
- Приём платежей через бота (QR-код, ссылка)
- Автоматическое формирование чеков (54-ФЗ)
- Налоговый калькулятор (авторасчёт налога "Мой налог")
- Отчётность — автозаполнение декларации
- Клиентская база

## 5. Техническая архитектура

### Стек технологий

**Frontend:**
- Next.js 14 (App Router)
- Tailwind CSS
- Telegram Mini App (основной канал)
- React Native (мобильное приложение — позже)

**Backend:**
- Node.js + NestJS
- PostgreSQL
- Redis

**AI:**
- OpenAI API (GPT-4)
- RAG система (поиск по базе налоговых законов)
- Своя база знаний (НК РФ, письма ФНС, судебная практика)

**Интеграции:**
- Сбербанк API
- Тинькофф API
- Альфа-Банк API
- Контур.Экстерн
- ЮKassa, CloudPayments

### Структура системы
```
Telegram Bot / Web App / Mobile App
         │
    NestJS API Gateway
         │
    PostgreSQL + Redis
         │
    AI Engine (RAG + GPT-4)
```

## 6. Roadmap и этапы запуска

### Этап 1: MVP (Месяцы 1-3)
